# Typed TodoMVC Hyperapp

Experiment in typings for Hyperapp, with a TodoMVC implementation as demo/sandbox.

Try it out by:

```sh
git clone https://github.com/zaceno/zx-typed-todomvc-hyperapp.git
cd zx-typed-todomvc-hyperapp
npm i
npm start
# then open your browser at https://localhost:8080/ for a live 
# preview. Then start hacking away
```
